# Autocomplete and Autocorrect

## Objective

The objective of this project is to build a simple autocomplete and autocorrect system using NLP.

## About Dataset

This dataset contains millions of news headlines collected from ABC News. It is commonly used for Natural Language Processing (NLP) tasks such as text prediction, autocomplete and autocorrect.

## Tools Used

- Python
- Pandas
- NLTK
- SpellChecker
- Matplotlib

## Project Steps

- Load Dataset
- NLP Preprocessing
- Build Autocomplete Model
- Build Autocorrect Model
- Performance Evaluation
- Data Visualization

## Results

The model predicted next words and corrected misspelled words with good accuracy.
