# Sentiment Analysis

## Objective

The objective of this project is to analyse customer opinions using sentiment analysis.

## About Dataset

This dataset contains customer reviews and their sentiments. It helps identify whether a review is positive, negative or neutral.

## Tools Used

- Python
- Pandas
- NLTK
- TextBlob
- Matplotlib

## Project Steps

- Load Dataset
- Text Preprocessing
- Sentiment Analysis
- Data Visualization
