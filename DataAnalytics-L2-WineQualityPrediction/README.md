# Wine Quality Prediction

## Objective

The objective of this project is to predict wine quality using machine learning.

## About Dataset

This dataset contains chemical properties of wine such as acidity, sugar, pH and alcohol content. These features are used to predict wine quality.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Steps

- Load Dataset
- Data Cleaning
- Exploratory Data Analysis
- Model Training
- Model Evaluation

## Results

The model predicted wine quality based on different chemical features.
