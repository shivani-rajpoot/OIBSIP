# Credit Card Fraud Detection

## Objective

The objective of this project is to detect fraudulent credit card transactions.

## About Dataset

This dataset contains credit card transaction records with normal and fraudulent transactions. It is used to build a fraud detection model.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Steps

- Load Dataset
- Data Cleaning
- Exploratory Data Analysis
- Model Training
- Fraud Detection

## Results

The model identified fraudulent transactions with good accuracy.
