# Google Play Store Analysis

## Objective

The objective of this project is to analyse Google Play Store apps and user reviews.

## About Dataset

This dataset contains information about Android apps such as app name, category, rating, reviews, installs, price and user sentiment. It is used to analyse app performance and customer feedback.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## Project Steps

- Load Dataset
- Data Cleaning
- Exploratory Data Analysis
- Sentiment Analysis
- Data Visualization

## Results

This project analysed app ratings, installs and user reviews to find useful insights.
