# EDA on Retail Sales Data

## Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on a retail sales dataset and find useful business insights.

## Dataset

Retail Sales Dataset

## Tools and Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow

- Data Loading
- Data Cleaning
- Missing Value Handling
- Duplicate Check
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insights

## Visualizations

- Sales Distribution
- Product Category Analysis
- Monthly Sales Analysis
- Correlation Heatmap

## Conclusion

This project helps understand customer purchasing behavior, sales trends, and product performance using data analysis and visualization.
