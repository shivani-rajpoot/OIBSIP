# House Price Prediction

## Objective

The objective of this project is to predict house prices using Linear Regression.

## About Dataset

This dataset contains house information such as area, bedrooms, bathrooms, location and price. It is used to predict house prices based on different features.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Steps

- Load Dataset
- Data Cleaning
- Exploratory Data Analysis
- Model Training
- Price Prediction
- Model Evaluation

## Results

The model successfully predicted house prices using Linear Regression.
